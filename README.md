PSToExe (PowerShell to EXE Converter)
A modern, high-performance graphical utility designed to bundle, compile, and deploy PowerShell scripts (.ps1) into standalone Windows executables (.exe). Built with a clean, centered interface, PSToExe bridges the gap between raw scripting and secure, seamless corporate software deployment.

🚀 Key Features
Advanced GUI & Center-Focused Design: A polished, modern desktop layout designed for clarity, ensuring all text, input fields, and action modules are perfectly aligned and optimized for high visibility.

Persistent File Tracking & Session Memory: The application intelligently remembers previously uploaded files and conversion profiles, allowing you to quickly reload, modify, or recompile past projects without starting from scratch.

Integrated PDF Uploader: Built-in capabilities to upload and link documentation or reference guides straight into the compilation workspace.

Deterministic File & State Management: * Features a robust Delete/Remove feature that safely clears temporary assets and history logs.

Implements precise scope encapsulation using private script blocks and .GetNewClosure(). This guarantees that UI removal hooks dynamically map to the correct immutable file paths, avoiding memory leaks or button-action collisions.

Payload Security: Packages scripts into executable containers, making it easier to distribute automated tools to end-users without exposing underlying code or triggering execution policy headaches.

🛠️ Workspace & Architecture
Engine: PowerShell Automation & UI Rendering Core.

Target OS: Windows 10 / 11 / Windows Server.

State Management: Isolated, closures-protected memory blocks to ensure stable multi-file handling and predictable cleanup routines.

📖 How To Use
Launch the Application: Run the primary script to launch the standalone graphical control center.

Import Script: Select your target .ps1 file via the file selection module.

Include Documentation (Optional): Use the integrated PDF loader to attach deployment documentation or README guides.

Compile: Generate your standalone, distribution-ready .exe.

Manage History: Use the persistent project list to revisit past compilations or cleanly remove them using the secure delete function.
